# Smart File Organizer 📁

A simple Python desktop application built with Tkinter that organizes files automatically based on their extensions.

---

## 🚀 Features
- Select any folder
- Automatically categorize files
- Move files into folders
- Live activity log
- File analysis report

---

## 🛠 Technologies
- Python
- Tkinter
- os
- shutil
